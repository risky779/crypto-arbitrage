# 반자동 차익거래 봇 사용 가이드

## 🚀 시작하기

### 1단계: API 키 설정

1. `.env.example` 파일을 `.env`로 복사
2. 업비트에서 API 키 발급:
   - https://upbit.com/mypage/open_api_management 접속
   - "Open API 사용하기" 클릭
   - **중요: "출금" 권한은 비활성화하세요!**
   - "자산 조회", "주문 조회", "주문하기" 권한만 활성화
3. `.env` 파일에 API 키 입력:
   ```
   UPBIT_ACCESS_KEY=your_access_key
   UPBIT_SECRET_KEY=your_secret_key
   ```

### 2단계: 거래 설정 조정

`.env` 파일에서 다음 설정을 조정하세요:

```
MIN_PROFIT_RATE=1.5           # 최소 수익률 (%) - BCH 기준 1.5~2.0 추천
MAX_TRADE_AMOUNT_KRW=100000   # 1회 최대 거래 금액 - 소액으로 시작 추천
MAX_DAILY_TRADES=10           # 일일 최대 거래 횟수
```

### 3단계: 봇 실행

```bash
# 방법 1: 배치 파일로 실행
start_trading_bot.bat

# 방법 2: 직접 실행
python trading_bot.py
```

## 📊 작동 방식

1. **기회 탐지**: 실시간으로 차익거래 기회 모니터링
2. **상세 분석**: 수익률, 수수료, 예상 이익 계산
3. **사용자 승인**: 거래 상세 정보를 보여주고 승인 요청
4. **자동 실행**: 승인 시 자동으로 매수/전송/매도 실행
5. **결과 기록**: 모든 거래 내역 로그 저장

## ⚠️ 안전 수칙

### 필수 사항
- ✅ 소액으로 테스트 먼저 (10만원 이하)
- ✅ API 키에서 "출금" 권한 비활성화
- ✅ 각 거래 승인 전 꼭 확인
- ✅ 일일 거래 한도 설정

### 주의 사항
- ⚠️ 시장 변동성으로 손실 가능
- ⚠️ 네트워크 지연 시 가격 변동
- ⚠️ 입출금 시간 동안 가격 변화
- ⚠️ 거래소 점검 시간 확인

## 🔧 다음 단계

### 현재 상태: 시뮬레이션 모드
- 실제 거래는 실행하지 않음
- 거래 흐름만 시뮬레이션

### 실제 거래 활성화하려면:
1. `trading_bot.py`의 `execute_trade` 함수 구현 필요
2. 충분한 테스트 후 활성화
3. 소액으로 시작하여 점진적 확대

## ❓ 문제 해결

### API 키 오류
- 키를 정확히 복사했는지 확인
- 권한 설정 확인 (자산/주문 조회, 주문하기)
- IP 화이트리스트 설정 확인

### 거래 실패
- 잔고 부족 확인
- 최소 주문 금액 확인
- 거래소 점검 시간 확인
