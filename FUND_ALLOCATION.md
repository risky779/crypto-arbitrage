# 💰 차익거래 자금 배치 가이드

## 🎯 핵심 원리

차익거래는 **양쪽 거래소에 모두 자금이 있어야** 합니다!

```
OKX (해외)          →  매수용 (USDT)
업비트 (국내)       →  매도용 (KRW) + 받을 공간
```

## 🔄 거래 방향별 필요 자금

### A. OKX → 업비트 (가장 흔한 경우)

```
OKX가 싸고 업비트가 비쌀 때
→ OKX에서 매수, 업비트에서 매도
```

**필요한 자금:**
- ✅ **OKX에 USDT** (매수용)
- ✅ **업비트에 KRW** (받은 BCH 매도용 공간 필요)

**예: 5만원 거래**
- OKX: 50,000원 상당 USDT (34 USDT)
- 업비트: 0원 이상 (공간만 있으면 됨, 하지만 여유금 권장)

---

### B. 업비트 → OKX (드문 경우)

```
업비트가 싸고 OKX가 비쌀 때
→ 업비트에서 매수, OKX에서 매도
```

**필요한 자금:**
- ✅ **업비트에 KRW** (매수용)
- ✅ **OKX에 USDT** (받은 BCH 매도용 공간 필요)

## 💡 실전 준비 단계

### Step 1: 원화 준비
```
총 자금을 결정 (예: 150,000원)
→ 절반씩 나눔 (각 75,000원)
```

### Step 2: OKX 입금
```
1. OKX 로그인
2. 자산 → 입금
3. P2P 거래 또는 카드 구매로 USDT 구매
4. 약 51 USDT 확보 (75,000원 상당)
```

### Step 3: 업비트 입금
```
1. 업비트 로그인
2. 입출금 → KRW 입금
3. 가상계좌로 75,000원 입금
4. 즉시 반영됨
```

## 📋 구체적 입금 가이드

### OKX USDT 구매 방법

**방법 1: P2P 거래 (추천)**
```
1. OKX → 거래 → P2P
2. USDT 구매 선택
3. KRW로 거래할 판매자 선택
4. 계좌 이체 후 USDT 받음
5. 수수료: 0~1%
```

**방법 2: 카드 구매**
```
1. OKX → 구매 → 카드로 구매
2. USDT 수량 입력
3. 카드 정보 입력
4. 즉시 입금 (수수료 약 3~5%)
```

**방법 3: 다른 거래소에서 전송**
```
1. 바이낸스/업비트 등에서 USDT 보유 시
2. OKX USDT 입금 주소 복사
3. 다른 거래소에서 출금
4. 수수료: 네트워크 수수료만 (약 1~2 USDT)
```

---

### 업비트 KRW 입금

**매우 간단!**
```
1. 업비트 앱/웹 로그인
2. 입출금 → KRW → 입금
3. 가상계좌 번호 확인
4. 은행 앱에서 해당 계좌로 이체
5. 1~2분 내 즉시 반영
```
