## 🏦 업비트 입금 (5분)

2. **입금 메뉴**

   - 은행 앱에서 해당 계좌로 이체
   - 금액: 75,000원 (또는 원하는 금액)

5. **완료**
   - 1~2분 내 업비트에 반영
   - 입출금 → KRW에서 확인

## 🌐 OKX 입금 (10~20분)

### 방법 1: P2P 거래 (추천)

**장점:**
- 수수료 거의 없음 (0~1%)
- 안전 (OKX 에스크로)

**단계:**

1. **OKX P2P 접속**
   ```
   OKX 로그인 → 거래 → P2P 거래
   ```

2. **구매 설정**
   - 통화: USDT 선택
   - 결제: KRW 선택
   - 금액: 51 USDT 입력
   - "구매" 클릭

3. **판매자 선택**
   - 평점 높은 판매자 (98% 이상)
   - 거래 횟수 많은 사람
   - 완료율 높은 사람

4. **주문 생성**
   - 판매자의 은행 계좌 정보 확인
   - 제시된 금액 확인
   - "구매" 버튼 클릭

5. **송금**
   - 판매자 계좌로 KRW 이체
   - 15분 내 이체 완료 필요

6. **확인**
   - "송금 완료" 버튼 클릭
   - 판매자가 확인하면 USDT 받음
   - 5~10분 소요

---

### 방법 2: 카드 구매 (빠름)

**장점:**
- 즉시 입금 (1~2분)
- 간단함

**단점:**
- 수수료 높음 (3~5%)

**단계:**

1. **OKX 구매 메뉴**
   ```
   OKX 로그인 → 구매 → 카드로 구매
   ```

2. **금액 입력**
   - USDT 51개 입력
   - 또는 KRW 75,000원 입력

3. **카드 정보**
   - 카드 번호
   - 유효기간
   - CVV

4. **결제**
   - "구매" 버튼
   - 카드 인증 (SMS/앱)

5. **완료**
   - 1~2분 내 USDT 입금
