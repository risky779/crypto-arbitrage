# 바이낸스-업비트 김프 모니터링 시스템

## 개요
바이낸스(USDT 마켓)와 업비트(KRW 마켓) 간 코인 가격 차이를 실시간으로 모니터링하여 차익거래 기회를 포착하는 시스템

## 모니터링 대상 코인 (13개)
- 메이저: BTC, ETH, XRP, SOL
- 알트코인: ADA, DOGE, AVAX, DOT, LINK, UNI, ATOM, BCH, SAND

## 수수료 계산
- 바이낸스 거래 수수료: 0.1%
- 업비트 거래 수수료: 0.05%
- 입출금 수수료: 0.1% (대략)
- **총 수수료: 0.25%**

실제 차익 = 김프율 - 0.25%

## 실행 방법

### 1회 테스트
```bash
python test_monitor.py
```

### 실시간 모니터링 (10초 주기)
```bash
python arbitrage_monitor.py
```

또는 배치 파일 더블클릭:
```
start_monitor.bat
```

종료: `Ctrl + C`

## 출력 예시
```
[바이낸스-업비트 김프 모니터링] 2026-05-28 15:32:31
환율: 1 USDT = 1,474.00 KRW
================================================================================

[+] DOT
  바이낸스: $        1.18 USDT  (       1,745 KRW)
  업비트:          1,750 KRW
  김프:       0.27%
  순이익:     0.02% (수수료 0.25% 차감)
```

- `[+]`: 순이익 +0.5% 이상 (기회 가능)
- `[ ]`: 중립 (-0.5% ~ +0.5%)
- `[-]`: 순이익 -0.5% 이하 (역김프)

1% 이상 차익 발생 시 `*** 차익거래 기회!` 메시지 표시
