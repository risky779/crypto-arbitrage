## 📋 다음 단계 (순서대로)

### Step 1: API 키 발급 (5분)

1. **업비트 로그인**
   - https://upbit.com 접속 및 로그인

2. **Open API 관리**
   - 우측 상단 프로필 → "Open API 관리" 클릭
   - https://upbit.com/mypage/open_api_management

3. **API 키 생성**
   - "Open API 사용하기" 클릭
   - 권한 설정:
     - ✅ 자산 조회
     - ✅ 주문 조회  
     - ✅ 주문하기
     - ❌ **출금 (반드시 비활성화!)**
   - IP 주소 제한 (선택사항, 보안 강화)
   - OTP 인증 완료

4. **키 저장**
   - Access Key와 Secret Key 안전하게 복사
   - **절대 타인과 공유하지 마세요!**

### Step 2: 환경 설정 (2분)

```bash
# 1. .env 파일 생성
copy .env.example .env

# 2. .env 파일 열고 API 키 입력
notepad .env
```

`.env` 파일 내용:
```
UPBIT_ACCESS_KEY=여기에_Access_Key_입력
UPBIT_SECRET_KEY=여기에_Secret_Key_입력

MIN_PROFIT_RATE=2.0              # BCH 기준 2% 이상 권장
MAX_TRADE_AMOUNT_KRW=50000       # 처음엔 5만원으로 시작
MAX_DAILY_TRADES=5               # 하루 최대 5회
```

### Step 3: 테스트 실행 (3분)

```bash
# 거래 봇 실행
start_trading_bot.bat
```

**첫 실행 시:**
- API 연결 확인
- 잔고 조회 테스트
- 시뮬레이션 거래 실행

### Step 4: 실전 준비 (선택사항)

**자금 준비:**
- 업비트에 KRW 입금 (추천: 10~50만원으로 시작)
- 소액으로 충분히 테스트 후 증액

**모니터링:**
- BCH 집중 모니터링 (가장 높은 수익률)
- 2% 이상 기회 포착 시 알림

## ❓ FAQ

**Q: 정말 안전한가요?**
A: 반자동 모드라 모든 거래를 승인해야 실행됩니다. API 출금 권한도 비활성화합니다.

**Q: 얼마로 시작해야 하나요?**
A: 5~10만원으로 시작하여 익숙해지면 증액하세요.

**Q: 하루에 얼마나 벌 수 있나요?**
A: 시장 상황에 따라 다르지만, BCH 기준 10만원당 거래 1회에 1,500~2,500원 정도입니다.

**Q: 손실 가능성은?**
A: 가격 변동, 슬리피지, 전송 지연 등으로 손실 가능합니다. 소액으로 시작하세요.

**Q: 24시간 켜둬야 하나요?**
A: 기회가 많은 시간대(오전 9시~11시, 저녁 8시~10시)에만 실행해도 됩니다.
